Gmail
Connect your Gmail account to read, archive and send emails. Requires a Google Cloud OAuth2 app.
How to set up Google OAuth2 credentials
  1. Go to console.cloud.google.com → create a new project
  2. Enable the Gmail API (APIs & Services → Library)
  3. Go to APIs & Services → Credentials → Create OAuth client ID
  4. Application type: Web application
  5. Authorized redirect URIs: add
  6. Copy the Client ID and Client Secret below
  7. In OAuth consent screen, add your Gmail address as a test user
Google Tasks
Connect Google Tasks to enable the courses sync (Keep ↔ Tasks). Uses the same OAuth credentials as Gmail — no extra setup needed.

Make sure https://app.opennoclaw.com/api/google-tasks/callback is added as an authorized redirect URI in your Google Cloud OAuth client, and that the Tasks API is enabled.
